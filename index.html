My Projects

Project 1:Hospital Management System

Hospital Management System A comprehensive Hospital Management System built using Java 17, Spring Boot, Spring MVC, Spring Data JPA, Thymeleaf, MySQL, and Maven. The system allows hospitals to efficiently manage patient records, doctor schedules, appointments, and billing information. The key features include:
Patient Management: CRUD operations for patient records, including medical history and appointments. Doctor & Staff Management: Scheduling and managing doctor availability and assignments. Billing Module: Automated billing generation with detailed reports.
Role-Based Access Control: Admin, doctors, and staff have different access privileges to the system. Responsive Design: User-friendly interface with Thymeleaf templates for a seamless experience across devices.

Project 2:Milk Billing System

A Milk Billing System built using Java 17, Spring Boot, Spring MVC, Spring Data JPA, MySQL, Thymeleaf, Maven, and Eclipse STS. This project streamlines the billing process for dairy businesses, enabling them to manage customer data, track daily milk deliveries, and generate monthly invoices. Key features include:
Customer Management: Add, update, and delete customer details, including milk delivery preferences.
Daily Delivery Tracking: Track daily milk delivery quantities for each.
Invoice Generation: Automatically calculate and generate monthly bills based on delivery records.
Search & Filter: Easily search for customers and filter records by date or amount.
Dashboard: Overview of daily deliveries, total sales, and pending payments for better business insights.

Project 3: E-Commerce Backend API

An E-Commerce Backend API developed using Java, Spring Boot, Hibernate, and MySQL, focusing on creating a scalable backend architecture. This project includes the core functionalities needed for an online store, like product management, order processing, and user authentication. Key features include:
Product Management: CRUD operations for product listings with features like category management, price adjustments, and stock tracking.
Order Management: Handle customer orders, payment processing, and order status updates.
User Authentication: Secure login and registration using JWT-based authentication.
API Documentation: Using Swagger to document RESTful APIs, making it easier for frontend developers to integrate.
Microservices Architecture: Utilized to separate different services like user management, inventory, and order processing for better scalability.
